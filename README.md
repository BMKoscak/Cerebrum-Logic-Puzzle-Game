Cerebrum: The Sequence
Cerebrum is a browser-based logic and pattern recognition game designed to test your mathematical intuition and critical thinking. Featuring over 100 procedurally generated levels with increasing complexity, players must solve sequences, code-breaks, and logic riddles to advance.
🎮 Features
* 100+ Levels: Difficulty scales from simple arithmetic to complex binary conversions and lateral thinking riddles.
* Checkpoint System: Progress is saved automatically every 10 levels.
* Lives & Integrity: You are allowed 5 mistakes per run. Failure results in a system reboot back to the last checkpoint.
* Hint System: Stuck? Use earned hints to decipher the pattern logic.
* Persistent Save: Your progress is stored locally in your browser.
* Responsive Design: Optimized for Desktop and Mobile play.
🚀 How to Play
1. Open the index.html file in any modern web browser.
2. Analyze the sequence or pattern shown on the main screen (e.g., 2, 4, 8, 16, ?).
3. Type your answer into the input terminal and press Enter or click EXECUTE.
4. Correct Answer: You advance to the next level. You gain a Hint every 5 levels.
5. Wrong Answer: Your System Integrity drops. If you make 5 mistakes, you revert to the previous Checkpoint.
🛠️ Installation
No installation required. The game runs entirely in a single HTML file with zero external dependencies (other than CDNs for styling).
1. Clone the repository:
git clone [https://github.com/BMKoscak/Cerebrum-Logic-Puzzle-Game](https://github.com/BMKoscak/Cerebrum-Logic-Puzzle-Game)

2. Navigate to the folder and open index.html.
